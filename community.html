Cowsay Development Community

GitHub

The main home for the cowsay project and codebase is on GitHub, under the umbrella of the cowsay-org organization. @apjanke is the Benevolent Dictator of cowsay-org.

Repos:

  • cowsay-org/cowsay: the main project repo
  • cowsay-org/cowsay-www: this web site

The main cowsay Issue Tracker is the main point of contact for cowsay support. If in doubt, post there.

Discussion Groups / Mailing Lists

Cowsay uses Google Groups as discussion forums and mailing lists.

There are three groups for cowsay:

  • cowsay-announce: a low-volume list that announces new releases and significant bugs
  • cowsay-developers: for developers of the cowsay codebase and project
  • cowsay-users: for questions and general discussions about cowsay
